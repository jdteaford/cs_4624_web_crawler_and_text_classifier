# cs_4624_web_crawler_and_text_classifier


Commands to install requirements for back end:

    cd /flask
    
    pip install Flask==3.0.2 pymongo==4.4.0 Flask-Cors==4.0.0 requests==2.31.0 PyJWT==2.8.0 flask_jwt_extended==4.6.0 numpy==1.24.4 pandas==2.0.3 gensim==4.3.2 tdqm==0.0.1 torch==2.2.2 scikit-learn==1.3.2 requests_html==0.10.0 lxml_html_clean==0.1.1 bs4==0.0.2


To run the back end:

    cd /flask
    python backend.py

To run the front end:

Run the following commands:

    cd client-final
    npm install
    npm run
